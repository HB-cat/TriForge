层级				核心协作关系
主控层 → 三线层	总理 Agent 将项目目标拆解为产品、设计、技术三条并行任务流，通过 SOLO 调度对应 Sub Agent
产品线 → 设计线	产品经理 Agent 产出 PRD 后，设计师 Agent 基于 PRD 中的页面结构和功能逻辑启动设计工作
产品线 → 技术线	产品经理 Agent 产出 PRD 后，技术经理 Agent 基于 PRD 中的业务规则和验收标准启动技术评审
设计线 → 技术线	设计师 Agent 产出设计规范和 Design Tokens 后，技术经理 Agent 将其纳入技术方案，作为前端实现的还原标准
共识与对冲		在上述三条协作链路上设置三个共识/对冲节点，确保所有冲突在技术执行前被识别和解决
技术线 → 测试		技术经理统管测试 Agent，开发完成后启动测试验收，测试结果作为上线决策依据
用户决策		测试/设计/产品三维验收 → 用户确认上线或进入迭代 → 总理发起复盘
